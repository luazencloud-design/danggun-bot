# 🥕 당근 자동응답 봇

당근마켓 판매자를 위한 AI 자동응답 챗봇 (Gemini 기반). 당근은 쇼핑몰이 아니라 이웃 간 중고거래 플랫폼이니, 챗봇이 할 일은 **구매자 문의에 짧고 자연스럽게 응대**하는 것뿐입니다. 주소·결제·송장은 당근페이가 알아서 처리해줘요.

## ✨ 핵심 기능

- **🔗 URL 자동 불러오기** — 당근 상품 URL을 붙여넣으면 상품명·가격·상태·구성품 등을 AI가 자동으로 채워줌
- **📋 텍스트 붙여넣기 백업** — URL 접근이 막힐 경우, 상품 설명 텍스트만 붙여넣어도 자동 분석
- **⚙️ 설정 기반 응대** — 가격·마지노선·네고 정책·거래 방법·말투를 미리 설정
- **💬 짧고 자연스러운 응답** — 1~2문장이 기본. 당근 판매자 톤으로
- **🚨 자동 수동 전환 (핸드오프)** — 계좌이체·카톡·전화번호·외부결제·사진요청·구매확정 등 위험 신호 감지시 봇이 즉시 정지하고 사람 확인을 요청
- **💾 로컬 저장** — 설정값이 브라우저에 자동 저장

## 🚀 실행 방법

### 사전 준비
1. [Node.js](https://nodejs.org) 18 이상 설치
2. [Gemini API 키](https://aistudio.google.com/app/apikey) 무료 발급 (`AIza...`로 시작)

### 실행 (가장 쉬움) ⭐

**`실행.bat` 파일을 더블클릭하세요.**

- 첫 실행시 의존성 자동 설치 (1~2분)
- 이후엔 바로 개발 서버 시작 + 브라우저 자동 열림
- 종료하려면 검은 창에서 `Ctrl+C`

### 실행 (명령줄)
```bash
cd danggeun-autobot
npm install     # 최초 1회
npm run dev
```

브라우저가 자동으로 `http://localhost:5173`으로 열립니다. 최초 실행 시 Gemini API 키 입력창이 나오니 발급받은 키(`AIza...`)를 넣어주세요.

## 📖 사용 방법

### 1단계: 상품 정보 불러오기 (3가지 방법 중 선택)

**방법 A: 당근 URL로 자동 불러오기** ⭐ 추천
1. 당근 앱에서 내 상품 → 공유 → 링크 복사
2. 설정 탭 상단의 **"URL"** 모드에서 링크 붙여넣기
3. **"불러오기"** 버튼 클릭 → 상품명·가격·상태 등이 자동으로 채워짐

**방법 B: 상품 설명 직접 붙여넣기**
1. URL 방식이 안 되면 **"직접 붙여넣기"** 탭으로 전환
2. 당근 상품 페이지의 제목·가격·설명을 복사해서 붙여넣기
3. **"텍스트에서 불러오기"** 클릭

**방법 C: 직접 입력**
- 각 필드를 직접 타이핑

### 2단계: 판매 정책 설정

- **가격·네고**: 판매가, 마지노선(최저가), 네고 정책(불가/소폭/협의)
  - URL로 불러온 경우 마지노선은 판매가의 97%로 자동 계산 (필요시 수정)
- **거래 방법**: 택배·직거래 각각 토글, 안내 문구
- **응대 스타일**: 말투(친근/정중/쿨), 추가 규칙

설정 후 **"설정 저장"** 클릭.

### 3단계: 채팅 테스트

상단 **"채팅 테스트"** 탭에서 구매자 입장에서 메시지를 보내 봇의 응답을 확인:

- "판매 중인가요?"
- "상태 어떤가요?"
- "네고 가능할까요?"
- "택배 가능해요?"
- "구매할게요"

빨강으로 표시된 테스트 버튼(`계좌번호 알려주세요`, `카톡으로 얘기해요`, `실물 사진 보내주세요`)을 누르면 수동 모드 전환이 작동하는 걸 볼 수 있어요.

## 🚨 핸드오프(수동 전환) 동작 방식

봇은 아래 신호를 감지하면 **즉시 일시 정지**하고 상단에 빨강 배너를 띄웁니다:

### 1단계 — 키워드 필터 (API 호출 전 차단)
- 계좌이체·무통장 입금
- 카톡·카카오톡·오픈채팅 등 외부 메신저
- 전화번호 (010-xxxx-xxxx)
- 외부/직접 결제
- 해외 배송
- 이메일·외부 링크
- 해외 결제 수단(페이팔·위챗·알리페이)
- 외부 플랫폼(중고나라·번개장터) 언급

### 2단계 — AI 판단 (맥락 분석)
- 실물/추가 사진 요청
- 판매자 개인정보 질문 (이름·나이·거주지 등)
- 구매 확정 의사 ("구매할게요", "결제할게요")
- 설정에 없는 스펙·히스토리 질문
- 분쟁·환불·AS 얘기
- URL 포함 메시지

### 수동 모드 UI
- 채팅창 **상단 빨강 배너** — 전환 사유 + "봇 재개" 버튼
- 사이드바 **봇 상태 카드** — 빨강 표시 + 재개 링크
- 채팅 피드 내 **시스템 버블** — 타임라인 기록
- **입력창은 비어있는 상태 유지** — 사용자가 실수로 엔터 눌러도 안전 (드래프트 주입 안 함)

봇이 일시 정지된 동안 구매자가 추가 메시지를 보내도 봇은 **사일런스** 상태로 응답하지 않아요. 사용자가 직접 타이핑해서 답하거나, "봇 재개" 버튼을 눌러야 자동 응답이 복귀합니다.

## ⚠️ 주의사항

**API 키 보안**
- API 키는 브라우저 LocalStorage에만 저장 (본인 PC 전용)
- 실제 웹서비스로 배포하려면 백엔드 서버 경유 필수

**비용 (Gemini 2.5 Flash 기준)**
- 일반 채팅: 메시지당 약 $0.0001 내외 (Claude 대비 1/10~1/20)
- URL 불러오기: Google Search grounding 포함으로 약간 더 높음
- [Google AI Studio](https://aistudio.google.com) 무료 할당량부터 시작 가능

**URL 불러오기 한계**
- 당근마켓이 봇 접근을 차단하는 경우 실패할 수 있음
- 이 경우 **"직접 붙여넣기"** 탭을 사용하세요

## 🛠️ 커스터마이징

- **모델 변경**: `src/App.jsx` 상단 `MODEL_CHAT`, `MODEL_IMPORT` 수정 (기본 `gemini-2.5-flash`)
- **응대 규칙**: `buildSystemPrompt` 함수 내부
- **위험 키워드 추가**: `SCAM_PATTERNS` 배열 (App.jsx 상단)
- **기본값**: `DEFAULT_CONFIG` 객체
- **마지노선 자동 계산 비율**: `applyToConfig` 함수의 `0.97` 값 조정
